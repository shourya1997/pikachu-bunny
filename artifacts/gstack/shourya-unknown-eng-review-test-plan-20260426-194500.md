## Affected Pages/Routes

- `/` or `/audit` - V1.0 audit workspace with import guide, dashboard, timeline, evidence drawer, and local-only status.
- `POST /api/imports` - local FastAPI endpoint for salary slip and EPF passbook imports.
- `GET /api/jobs/{job_id}` - local FastAPI endpoint for import progress, cancellation, success, and failure states.
- `GET /api/audits/{audit_id}` - local FastAPI endpoint for dashboard facts, findings, timeline months, and evidence snippets.

## Key Interactions to Verify

- Import one supported salary slip fixture and one supported EPF passbook fixture.
- Show API processing progress without blocking the web UI.
- Cancel an import job and verify no audit data changes.
- Show V1.0 dashboard states: clean, confirmed mismatch, needs review, unsupported, failed, and partial.
- Open the highest-priority finding by default.
- Navigate finding rows with keyboard focus.
- Open evidence drawer and verify masked identifiers.

## Edge Cases

- Missing file, empty file, unsupported file type, unsupported layout, invalid month, invalid amount, duplicate import, replaced source file, stale EPF passbook, parser failure, API job failure, storage write failure, corrupt local database, and user-cleared local data.
- Salary slip value `Rs 1,800.00` versus EPF value `1800` should normalize without creating a false mismatch.
- Employer spelling variation should produce needs-review only when identity confidence is insufficient.
- 24 months of salary slips with 3 missing EPF months and one duplicate row should produce 3 missing-month findings and ignore the duplicate safely.

## Critical Paths

- Supported salary slip + supported EPF passbook -> parse -> normalize -> store facts -> reconcile -> dashboard -> timeline -> evidence drawer.
- Missing second source -> partial state -> no confirmed mismatch.
- Unsupported source -> unsupported/failed state -> no confirmed mismatch.
- Confirmed mismatch -> evidence drawer -> masked identifiers -> factual next action.
- Local API unavailable -> UI shows clear local-engine error and recovery action.
